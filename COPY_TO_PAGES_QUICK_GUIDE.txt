================================================================================
HOW TO USE
================================================================================

1. Open document and select account/page
2. Add or edit fields on current page
3. Click "âœ“ Save" to save changes

5. Modal opens showing all other pages
6. Check the pages you want to copy to
7. Click "Copy Fields"
8. Success notification shows how many pages were updated

================================================================================
WHAT HAPPENS
================================================================================

When you copy fields from Page 1 to Page 2:
- Page 1 fields are merged with Page 2 fields
- Page 1 fields override Page 2 fields (if same name)
- New fields from Page 1 are added to Page 2
- Existing fields in Page 2 are preserved (unless overridden)
- Confidence scores are preserved
- Each page maintains independent data

Example:
  Page 1: Name, Email, Phone
  Page 2: Name, Address
  
  After copy:
  Page 2: Name (updated), Email (new), Phone (new), Address (preserved)

TECHNICAL DETAILS
================================================================================

Merge Logic:
  mergedData = { ...targetPageData, ...currentPageData }
  
  - Target page data is base
  - Current page fields override
  - New fields are added
  - Existing fields preserved (unless overridden)

API Calls:
  1. GET /api/document/{id}/account/{idx}/page/{num}/data
     â†’ Fetch target page data
  
  2. POST /api/document/{id}/account/{idx}/page/{num}/update
     â†’ Save merged data with action_type: 'copy'

Performance:
  - Time: O(n * m) where n = pages, m = fields
  - Network: 2n calls (1 fetch + 1 update per page)
  - UI: Non-blocking, responsive
