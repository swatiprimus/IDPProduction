Implementation Details

The flexible name matching algorithm:

  1. Normalize: Convert to uppercase, remove punctuation, trim whitespace
  2. Parse: Split into [First, Middle, Last] components
  3. Compare: First and Last MUST match, Middle is flexible
  4. Score: Calculate confidence based on match type
  5. Accept: Match if confidence >= 85%
