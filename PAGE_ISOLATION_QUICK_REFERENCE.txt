================================================================================
PAGE ISOLATION - QUICK REFERENCE
================================================================================

PRINCIPLE: Each page is completely independent

================================================================================
HOW IT WORKS
================================================================================

Page 1: Has its own fields
  - "abc" = "value1"
  - "name" = "John"

Page 2: Has its own fields (independent)
  - "address" = "123 Main St"
  - "phone" = "555-1234"

Page 3: Has its own fields (independent)
  - "date" = "2025-12-18"
  - "amount" = "1000"

Fields on Page 1 do NOT appear on Page 2 or Page 3

================================================================================
ADDING FIELDS
================================================================================

Add "abc" on Page 1:
  1. Navigate to Page 1
  2. Click "‚ûï Add"
  3. Enter: Field Name = "abc", Value = "value1"
  4. Click "Add Field"
  Result: "abc" appears on Page 1 ONLY

Add "abc" on Page 2:
  1. Navigate to Page 2
  2. Click "‚ûï Add"
  3. Enter: Field Name = "abc", Value = "value2"
  4. Click "Add Field"
  Result: "abc" appears on Page 2 ONLY
          Page 1 "abc" = "value1" (unchanged)
          Page 2 "abc" = "value2" (different value)

================================================================================
DUPLICATE DETECTION
================================================================================

Try to add "abc" again on Page 1:
  1. On Page 1
  2. Click "‚ûï Add"
  3. Enter: Field Name = "abc", Value = "new_value"
  4. Click "Add Field"
  Result: Confirmation dialog appears
          "Field 'abc' already exists. Do you want to overwrite it?"
          Click OK to overwrite or Cancel to keep original

Try to add "abc" on Page 2 (after adding on Page 1):
  1. Navigate to Page 2
  2. Click "‚ûï Add"
  3. Enter: Field Name = "abc", Value = "different"
  4. Click "Add Field"
  Result: NO error, field is added
          Page 2 now has "abc" = "different"
          Duplicate check is PER PAGE, not across pages

================================================================================

Each cache contains only that page's fields
